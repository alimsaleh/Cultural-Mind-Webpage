WELCOME TO CULTURAL MIND!

A place where you can expand your cultural knowledge about the world. Learn about local food, clothing, festivals, and places unique to each country. Enrich your travel experiences and the world will never cease to amaze you!

What is Cultural Mind?

Cultural mind is a site that was created to spread knowledge about cultures around the world. Every country has unrevealed secrets, and untold stories, and the site was built with this in mind. We wanted to help tell these stories. Users can learn about various cultural aspects of a country, and can even contribute themselves.

How it works

Below are screenshots and descriptions of the main features Cultural Mind offers.

Create an account

The first step to becoming a member of the site is creating an account. There is a simple create an account page where users can pick a username and choose a password. After users have done that, they can start posting.


Pick a destination

This is the explore page that was designed allow simple, and seamless navigation of countries. The main component on this page is an interactive world map that allows you to select individual countries. Countries are colored with different hues of blue that are influenced by the number of posts a country has. Also when users hover over a country they can see the number of posts for that country.

Browse all the posts for a country

All posts are public and there is no need to go through the hassle of creating an account just to view the posts. Individual posts have easily identifiable markers to let users know what the post is about.

View a post

View the full content of a post and learn more about a specific topic. The design of this page is minimalistic to allow for easy reading. Each post also has an image associated with it to help portray the information and increase user engagement.

Select a filter

Users can choose to focus on a particular cultural aspect of a country. The categories have distinct identifiable icons. Categories help users narrow down the content, so they can find exactly what they are looking for.

Create a post

Users can easily add posts if they want to contribute. They pick a title, select a country, provide a summary, a write up of their post, and an image for their post. This page also includes a simple rich text editor to help format posts in order to increase readability.
